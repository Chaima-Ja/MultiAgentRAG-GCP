# GCP Integration

This project integrates with multiple Google Cloud Platform services:

## Services Used

1. **Cloud Functions (Gen2)** - Serverless API endpoints
2. **Cloud Storage** - Document storage
3. **BigQuery** - Search logging and analytics
4. **Vertex AI** - LLM services (optional, can use Anthropic)
5. **Cloud Run** - UI hosting
